# Snowflame Example Apps

Small fixtures for testing workspace Serve.

## Examples

- `0` — publish `.`. One `index.html`.
- `1` — publish `.`. HTML with separate CSS and JS assets.
- `2` — publish `dist`. Already-built React-style static output.
- `3` — future build-mode fixture. Run `npm install` and `npm run build`, then publish `dist`.
- `4` — publish `.`. Small-business UI with Customers, Schedule, and Ledger tabs.

These examples are meant to be imported as workspace files and then published from the workspace Serve settings.
