# Snowflame Example Apps

Small fixtures for testing workspace Serve.

## Examples

- `single-file-html` — publish `.`. One `index.html`.
- `html-js-css` — publish `.`. HTML with separate CSS and JS assets.
- `built-react-app` — publish `dist`. Already-built React-style static output.
- `unbuilt-react-app` — future build-mode fixture. Run `npm install` and `npm run build`, then publish `dist`.

These examples are meant to be imported as workspace files and then published from the workspace Serve settings.
